Coffee Lover Project

This is a place to practice using Bootstrap

I am copying visual layout from another site using techniques I learnt online. Here are the steps I took:

  1. Created a genaral landing page with a jumbotron
  2. Next created a NavBar on the page that links to another html file
  3. Added two paragraphs of lorem below the jumbotron
  4. Used the grid system to add in pictures of coffee. Here are the image links:
    • One
    • Two
    • Three
    • Four
    • Five
    • Six
  5. Changed the grid system so the images collapse on smaller screen sizes
  6. Created another html file for the sign-up page
  7. Created a Form on this sign-up page with email, password and a check box.

There were some obstacles along the way. I also decided to include some extra features-

  • I enabled javascript and jquery to get the hamburger dropdown to work
  • I included a link to a page written using only html and css

That's it! Feel free to click around this site to test its functionality.
